Política de Privacidade

YouTube Shorts Blocker

✅ Nenhum dado coletado
Resumo

O YouTube Shorts Blocker não coleta, não armazena, não vende e não transmite nenhum dado pessoal do usuário. Toda a operação da extensão acontece localmente no seu navegador.

Dados coletados

O que a extensão acessa

A extensão acessa apenas o conteúdo necessário para realizar sua única função: bloquear os YouTube Shorts. Veja o que é e o que não é utilizado:

⚙️
Preferência de ativação (local) Armazenada apenas no seu navegador via chrome.storage.sync. Indica se o bloqueio está ligado ou desligado. Não sai do seu dispositivo.
🌐
URL da aba ativa do YouTube Verificada localmente para identificar páginas de Shorts e redirecionar para o player normal. Não é registrada nem transmitida.
🚫
Histórico de navegação Não coletado. A extensão não monitora quais vídeos você assiste.
🚫
Dados pessoais Nenhum. A extensão não acessa nome, e-mail, localização, conta do Google nem qualquer informação de identificação.
Permissões

Por que cada permissão é necessária

Permissão Motivo
activeTab Enviar mensagens entre o popup e o content script para ativar ou desativar o bloqueio sem recarregar a página.
storage Salvar a preferência do usuário (ligado/desligado) localmente entre sessões. Nenhum dado pessoal é salvo.
*.youtube.com Executar o script nas páginas do YouTube para detectar e ocultar elementos de Shorts e redirecionar URLs /shorts/.
Terceiros

Compartilhamento de dados

A extensão não compartilha nenhum dado com terceiros. Não há servidores externos, não há rastreamento e não há anúncios.

Código remoto

Todo o código está no pacote

A extensão não carrega nenhum JavaScript ou WebAssembly externo. Todo o código está incluído no pacote instalado e pode ser inspecionado diretamente.
